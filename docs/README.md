# Campaign Manager Dokumentation

Willkommen in der Dokumentation für den PnP Campaign Manager.

## Struktur

*   `test_cases/`: Enthält Testfälle für manuelle und (später) automatisierte Tests der Anwendung.
    *   `ui/`: Spezifische Anleitungen und Szenarien für den Test der grafischen Oberfläche (PyQt6).
