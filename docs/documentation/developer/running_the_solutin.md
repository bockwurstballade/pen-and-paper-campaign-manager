## How to Initialize the virtual environment

- Change into the main directory, e. g. `cd ~/pnp/pnp-campaign-manager`

```
cd ~/pnp/pnp-campaign-manager
python3 -m venv .venv
source .venv/bin/activate
pip install PyQt6
```

Run the executable

```
python3 ./campaign-manager.py
```

## Once the Virtual environment is built

Once the virtual environment in Python is built, you can short-circuit the procedure a little bit


```
cd ~/pnp/pnp-campaign-manager
source .venv/bin/activate
```

Run the executable

```
python3 ./campaign-manager.py
```
