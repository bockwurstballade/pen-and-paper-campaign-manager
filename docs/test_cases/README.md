# Test Cases

Dieser Ordner enthält alle definierten Test-Szenarien für die Anwendung.

## Ordnerstruktur

*   **`ui/`**: Testfälle, die das Durchklicken und Verifizieren der PyQt6 UI-Elemente beschreiben. Beispiele: "Charakter-Erstellung UI", "Kampf-Tracker Flow", etc.
*   *(Weitere Ordner wie `core/` oder `integration/` können hier bei Bedarf ergänzt werden)*
